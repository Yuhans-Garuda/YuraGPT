# YuraGPT - ChatGPT Exporter

Ekstensi browser untuk **backup / export** percakapan dari ChatGPT (chatgpt.com & chat.openai.com).
Semua fitur **gratis** (tanpa watermark & tanpa tracking).

## Fitur
- Download **Markdown (.md)**
- Download **HTML (.html)** (rapih + bisa diprint)
- Download **JSON (.json)**
- **Print / Save as PDF** menggunakan fitur **Print** bawaan browser (hasil PDF biasanya tetap bisa disalin teksnya)
- Nama file otomatis memakai **judul obrolan**
- Footer sosial (TikTok / Instagram / Threads) dengan handle **@yuhansnurfw** di tengah bawah halaman

## Cara pasang (Chrome / Edge)
1. Buka `chrome://extensions`
2. Aktifkan **Developer mode**
3. Klik **Load unpacked**
4. Pilih folder `YuraGPT`

## Cara pakai
1. Buka percakapan di ChatGPT
2. Klik ikon ekstensi → pilih format export
3. Untuk PDF: pilih **Print / Save as PDF**, lalu pada dialog Print pilih **Save as PDF**

## Privacy Policy
1. Data Collection
YuraGPT does not collect any personally identifiable information.
The extension does not:

Track users
Collect browsing history
Store chat content on external servers
Use analytics or advertising tools
All exported content remains on the user’s device.

2. Data Usage
The extension only accesses the current ChatGPT page content for the purpose of generating export files.
This data is processed locally inside your browser and is never transmitted elsewhere.

3. Data Sharing
YuraGPT does not share, sell, or transfer user data to any third parties.

4. Permissions
The extension may request permissions required to:
Read the active ChatGPT conversation
Generate and download export files
These permissions are used strictly for the extension’s core functionality.

5. Security
All processing happens locally. No chat data is uploaded or stored in any remote database, ensuring maximum privacy and security.

6. Changes to This Policy
This Privacy Policy may be updated in the future. Any changes will be posted on this page with an updated effective date.
